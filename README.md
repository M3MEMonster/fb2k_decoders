# fb2k_decoders

DISCLAIMER: ALL CODES (Excluding 3rdParty part) ARE WRITTEN BY CURSOR.

Some decoders mainly for module music for foobar2000.(Excluding Millisecond Displayer and Duration&Subsong Manager, they are utils.)

Millisecond Displayer is a visualization component that can display songlength and playtime in [H]:MM:SS.mmm

Duration&Subsong Manager can let users manage the songlength and subsongs in playlists.

All 3rdParty parts follow their own licenses.
